# From Automata to Undecidability: The Boundaries of Program Analysis

This repository contains the implementation and experimental demonstration for the conference paper.

## Contents
- Sample program with arithmetic error
- Static analysis tool output
- Conference paper PDF

## Tools Used
- Cppcheck
- Infer

## Purpose
This project demonstrates the practical limitations of program analysis tools and connects them to theoretical concepts such as undecidability and Rice’s theorem.

## How to Run
1. Compile the C program.
2. Run static analysis tools.
3. Observe warnings related to division by zero.
